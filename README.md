# RegisterProject

Скрипт автоматической регистрации пользователей для системы удалённого мониторинга параметров окружающей среды.

## Назначение

Скрипт предназначен для автоматической обработки заявок на регистрацию, поступающих через Google Forms. Он загружает CSV-файл с ответами, проверяет и при необходимости генерирует логин и пароль, создаёт системного пользователя Linux, настраивает FTP-доступ и отправляет регистрационное письмо на адрес электронной почты пользователя.

## Функциональные возможности

- Загрузка CSV-файла с ответами Google Forms (заявками пользователей на регистрацию в системе)
- Вычисление хеша строки для предотвращения повторной обработки одной заявки
- Проверка указанных пользователем в форме регистрации логина и пароля на соответствие требованиям:
  - Логин: длина ≥ 6 символов, только латиница и цифры
  - Пароль: длина ≥ 10 символов, хотя бы одна цифра, одна заглавная буква, один спецсимвол, без пробелов
- Автоматическая генерация логина (формат `userXXXXX`) и пароля (12 символов) при несоответствии требованиям пользовательских логина и пароля
- Создание системного пользователя Linux
- Создание домашней директории пользователя и папки `devices` для хранения данных его устройств (выносных узлов)
- Сохранение данных пользователя в JSON-файл (`users.json`)
- Отправка регистрационного письма с учётными данными через SMTP-сервер Mail.ru
- Ведение лога работы скрипта (`register.log`) с автоматической очисткой устаревших записей (старше 30 дней)

## Технологический стек

- **Язык программирования:** Python 3
- **Протоколы:** SMTP (отправка писем), HTTPS (загрузка CSV)
- **Системные вызовы:** useradd, chpasswd, chown, chmod, mkdir
- **Форматы данных:** CSV, JSON

## Установка и настройка

### Требования

- Операционная система Linux (Ubuntu/Debian рекомендуется)

- Доступ к интернету для загрузки CSV и отправки писем
- Права суперпользователя (sudo) для выполнения системных команд
